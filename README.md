# 灵感转瞬记录仪

一个极致快速的灵感捕捉工具，以语音为核心交互方式，帮助你在任何时刻、任何场景下捕捉转瞬即逝的创意火花。



## 创意背景

走路时、洗澡时、睡前、吃饭时，好想法常常突然冒出。但当你想记录时，要么手边没有工具，要么操作太慢——等准备就绪，80%的细节已经模糊。这个工具旨在解决这个痛点，让"想到"到"记下"的时间缩短到3秒以内。

## 核心功能

### 🔊 语音闪电记
- 主界面只有一个大按钮，点击即可开始录音
- 松手自动保存并转文字，从"想记录"到"记录完成"不超过3秒
- 支持空格键快捷操作，无需鼠标

### 🤖 AI 自动标签与摘要
- 智能识别内容类型（灵感/待办/备忘/情绪/书摘）
- 自动生成相关标签，便于后续检索
- 一句话概括录音内容，快速预览核心思想

### 💡 后续追问与深化
- 对已记录的灵感进行AI展开分析
- 提供多角度思考建议（用户视角、技术视角、商业视角）
- 让随手记的一句话有机会变成可执行的方案

### 📋 每日灵感回顾
- 每天固定时间推送今日灵感回顾
- 快速浏览当天记录，决定转化为待办、归档或深入思考
- 统计数据一目了然（今日记录、待办事项、已归档、待整理）

### 📱 响应式设计
- 支持桌面端和移动端
- 侧边栏导航，清晰的功能分区
- 适配各种屏幕尺寸

## 技术实现

### 前端技术栈
- **HTML5** - 语义化页面结构
- **CSS3** - 深色主题设计，流畅动画效果
- **JavaScript (ES6+)** - 模块化架构，面向对象设计

### 核心技术
- **MediaRecorder API** - 浏览器原生录音功能
- **Web Speech API** - 实时语音转文字
- **LocalStorage** - 本地数据持久化存储
- **AI 算法** - 基于关键词匹配的智能分类与标签生成

## 使用方法

### 本地运行
1. 克隆或下载项目到本地
2. 使用浏览器打开 `index.html` 文件
3. 允许浏览器访问麦克风权限
4. 点击大按钮或按空格键开始录音

### 快捷键
- **空格键** - 开始/停止录音
- **Esc** - 停止录音

### 注意事项
- 语音识别功能需要 HTTPS 环境或 localhost 才能正常工作
- 在非 HTTPS 环境下，会使用示例文本演示功能
